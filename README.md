# Fairdata Development Docker containers.

This repository contains containers which will help with development tasks.

## Usage
```
make up
```

### Metax dev
```
make metax-dev
```
If you need to access shell you can use
```
make metax-shell
```

### Qvain dev
```
make qvain-dev
```
If you need to access shell you can use
```
make qvain-shell
```
